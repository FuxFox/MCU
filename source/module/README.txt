* The files in this folder are Module layer code and Middleware.

* You can modify the code to support other MCU.

*list:
- md_battery	        -	The battery detection module
- md_beep		-	A beep module implemented base on NRF52832, provide advanced control of buzzer.
- md_clock		-	This is a clock module, with alarm function.
- md_lcd		-	This is a LCD display module
- md_led_pwm	        -	This is a led control module implemented base on NRF52832, with PWM.
- md_m26		-	QUECTEL M26 IOT module 
- md_yaohe		-	A Specific application of QUECTEL M26

- module	        -	module layer
